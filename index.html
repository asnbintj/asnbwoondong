사용자 권한 · 정보 관리

📖 사용 안내

궁금한 항목을 클릭하면 자세한 설명이 펼쳐집니다.

  • 로그인 — 이름(아이디)과 비밀번호를 입력해 로그인합니다. 기본 비밀번호는 1234입니다.
  • 로그인 정보 저장 — 로그인 화면의 "로그인 정보 저장" 체크박스를 체크하면 다음에 자동으로 입력됩니다.
  • 비밀번호 변경 — 매니저가 🔐 권한 관리 탭에서 각 사용자의 정보를 수정할 수 있습니다.
  • 역할 구분 — 매니저는 선수 추가·수정·삭제, 팀 나누기, 능력치 수정 요청 승인이 가능합니다. 일반 선수는 자신의 정보 조회와 능력치 수정 요청만 가능합니다.

모든 능력치는 0 ~ 50 범위이며 5 단위로 조정됩니다.

⚽ 축구 능력치
🦵 슛
골문을 향해 정확하고 강하게 슈팅하는 능력. 공격수에게 핵심 스탯입니다.
⚡ 스피드
드리블·압박·역습 시 빠르게 움직이는 능력. 윙어와 풀백에게 중요합니다.
🧠 패스
정확한 패스로 공격을 연결하는 능력. 미드필더의 핵심 스탯입니다.
🛡 수비
상대 공격을 차단하고 볼을 빼앗는 능력. 수비수에게 필수입니다.
💪 체력
경기 내내 높은 강도를 유지하는 지구력. 모든 포지션에 영향을 줍니다.
⚾ 야구 능력치
💥 타격
투구를 정확하게 치는 배팅 능력. 타율과 장타력에 영향을 줍니다.
🎯 투구
정확하고 빠른 공을 던지는 능력. 투수 포지션의 핵심 스탯입니다.
🧤 수비
타구를 잡고 정확하게 송구하는 능력. 내야·외야 모두에게 중요합니다.
🏃 주루
베이스를 빠르게 달리고 도루하는 능력. 기동력을 결정합니다.
💪 체력
경기 내내 집중력과 체력을 유지하는 능력. 모든 포지션에 영향을 줍니다.
💡 별명 자동 생성 — 가장 높은 능력치를 기준으로 별명이 자동으로 붙습니다. (예: 슛이 가장 높으면 "스나이퍼", 체력이 높으면 "강철체력" 등) 원하면 직접 별명을 설정할 수도 있습니다.
🏅 운동을 꾸준히 하면 능력치가 달라집니다! 실력이 늘었거나 현재 능력치가 실제와 다르다고 느끼면 수정을 요청할 수 있습니다. 매니저가 검토 후 반영합니다.
  1. 📋 선수 관리 탭에서 자신의 선수 카드를 클릭합니다.
  2. 선수 상세 정보 모달에서 📝 능력치 수정 요청 버튼을 클릭합니다.
  3. 원하는 능력치 값으로 슬라이더를 조정합니다. (현재 값과 같으면 요청 불가)
  4. 요청 사유를 간략히 입력하고 📤 요청 제출을 누릅니다.
  5. 매니저가 🔐 권한 관리 탭에서 요청을 확인하고 승인 또는 거절합니다.
  6. 승인되면 능력치가 즉시 반영됩니다. 🎉
✅ 언제 요청하면 좋을까요?
운동 모임에 꾸준히 참석하여 실력이 향상되었을 때, 또는 자신의 특기 포지션이 바뀌었을 때 요청해 보세요. 실제 플레이를 본 매니저가 판단하여 반영합니다.
  1. ⚽ 팀 나누기 탭으로 이동합니다. (매니저만 가능)
  2. 종목(축구/야구)과 팀 수(2~4팀)를 선택합니다.
  3. 참가할 선수들을 선수 목록에서 클릭하여 선택합니다.
  4. 팀 나누기 버튼을 누르면 남녀 성비와 능력치를 균형 있게 맞춰 팀을 구성합니다.
  5. 결과 이미지를 클립보드에 복사하거나 카카오톡 등으로 공유할 수 있습니다.
💡 팀 나누기 알고리즘 — 남자끼리, 여자끼리 능력치 순으로 정렬한 후 스네이크 드래프트 방식으로 배분합니다. 성비와 실력이 최대한 균등하게 맞춰집니다.
  • 내보내기 — 📋 선수 관리 탭 상단의 📥 엑셀 내보내기 버튼으로 현재 선수 전체 데이터를 xlsx 파일로 저장합니다.
  • 서식 파일 다운로드 — 📋 서식 다운로드 버튼으로 빈 양식을 받아 새 선수를 입력할 수 있습니다.
  • 가져오기 — 📤 엑셀 가져오기 버튼으로 작성한 파일을 업로드합니다. 미리보기 후 최종 확정합니다.
  • 이미 같은 이름의 선수가 있으면 덮어쓰지 않고 건너뜁니다.
